High level
[Edit]

Tuxedo is a middleware currently marketed by Bea. Complete online documentation is on http://edocs.bea.com/tuxedo/tux91/index.htm. For the integration Tuxedo version 9.1 (on Linux) has been used but older version was reported as working as well. Qore version used for the development was 0.5.2 and up.


Low level layer and high level (using the low level) are provided. The high level is recommended, the low level should be used for tasks not covered by the higher level layer. It is possible to use both levels at the same time.
